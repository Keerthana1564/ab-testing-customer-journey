## Objective

To evaluate whether device type affects customer purchase conversion using A/B testing.

## Dataset

Customer journey dataset containing user behavior and purchase outcomes.

## Methodology

* Split users into Desktop (control) and Mobile (test)
* Calculated conversion rates
* Performed t-test hypothesis testing
* Computed confidence intervals
* Visualized results

## Results

The Mobile group showed a higher conversion rate.
Statistical testing determined whether the difference was significant.

## Business Recommendation

Based on the statistical findings, recommendations were made to improve product strategy.

A/B testing enables data-driven decision-making in product optimization.
